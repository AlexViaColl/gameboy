# GameBoy Emulator in C

## Dependencies
| OS / Distro           | Packages                |
|-----------------------|-------------------------|
| Arch-based (Manjaro)  | pkgconf sdl2            |
| Debian-based (Ubuntu) | pkg-config libsdl2-dev  |
| Fedora-based          | pkgconf-pkg-config SDL2 |

## Quick Start
```console
$ ./build.sh
$ ./gb <path to ROM>
```
